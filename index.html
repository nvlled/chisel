Tutorial 1: Class selectors

Select the red box

Each box has a class that corresponds to their color. For example, a red box has a class of "red", and a blue box has a class of "blue".

To select the red box, enter .red in the input field and press [enter]

Tutorial 2: Class selectors

Select the blue box

To select the blue box, enter .blue in the input field and press [enter]

Tutorial 3: Class selectors

Select the green box

To select the green box, enter .green in the input field and press [enter]

Tutorial 4: Class selectors

Select all the blue boxes

A class can be applied to more than one element. A selector of .red will select all the red boxes. Enter .blue in the input field and press [enter]

Tutorial 5: ID selectors

Select the alpha red box

Whereas "class" is used for elements that have common properties, "id" is used to uniquely identify an element within a page.
The text inside the boxes corresponds to their id. In this case, one red box has an id "alpha", while the blue box has an id "bravo".

Try entering #bravo first to see the blue box selected.
After that, Enter #alpha in the input field to select the first red box.

Tutorial 6: Adjacent selectors

Select all the red boxes that are preceded by a blue box

The symbol + indicates an element adjacent with respect to another element. For example, X + Y will select all the element Y that has an X before it.

To achieve the current goal, enter the following:  .blue + .red

Tutorial 7: Adjacent selectors

Select the green box that is preceded by a red box

You know what to do by now right?
The green after a red is red plus green.

Tutorial 7.1: Adjacent selectors

Select the red box after charlie

You've seen adjacent selectors with class. You can also do it with id selectors. For instance, to select the blue box after alpha, you do: #alpha + .blue

Tutorial 7.2: More adjacent selectors

Select the red box preceded by two blue boxes

You are not limited to specifying to just two adjacent elements. You can, for example, select the red box preceded by two red boxes by
the selector: .red + .red + .red

Sum of two blues and one red reaches the goal.

Tutorial 9: Grouping selectors

Select green and blue boxes

Use a comma when using more than one selectors.
.green, .red will select both green and red while .red, blue will select red and blue.

To achieve the current goal, enter the following: .green, .blue

Note: You are free to explore. Try typing different combinations and see what happens.

Tutorial 10: Grouping selectors

Select all the blue and red boxes

Since class selectors will select more than one class, entering .green, .blue will select all, not just one, green and blue boxes.

Tutorial 11: Grouping selectors

Select alpha, bravo, and charlie

Any selectors can be grouped, not just classes, but also id selectors and others. Also, any number of selectors can be grouped, not just two.

For the goal, enter #alpha, #bravo, #charlie

Note: The order doesn't matter. #charlie, #alpha, #bravo or other permutations still achieves the same thing.

Tutorial 12: Grouping selectors

Tutorial 13: More grouping selectors

Select the blue box after charlie and all the green boxes

As mentioned earlier, any selectors can grouped. This includes the adjacent + selector.

This may be a tricky one. If you can't figure it out, mouseover on the following blacked text to reveal the answer: #charlie + .blue, .green
